Конспект с занятий

Словарик

p

p - paragraph - абзац. Тег для текста

h1..6

h - header - заголовок. Тег для создания загаловка (чем больше цифра, тем больше заголовок)

div

div - division - дробление. Универсальный тег, может содержать в себе другие теги, текстовую информацию

img

img - image - изображение. Тег для изображения. Имеет два атрибута:

src - source - источник, путь. Атрибут, адрес необходимого изображения

alt - alternative - альтернативный. Атрибут, альтернативный текст, который появится, если браузер не загрузит картинку
